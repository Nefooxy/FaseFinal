Página Inicial

Bem-Vindo a plataforma Aprenda Fácil, no nosso site você poderá iniciar uma jornada na programação com a linguagem HTML e CSS, oferecemos diversas opções de materias didáticos para que você escolha como prefere realizar seus estudos, além disso temos uma seção apenas com mídias digitais para você que tem maior facilidade para aprender dessa forma.
Nosso site também conta com a seção para a comunidade, que será uma página dedicada para a interação entre os usuários, na qual poderão postar suas dúvidas, responder as dúvidas postadas e formar grupos com o intuito de aprender cada vez mais e de forma mais leve.
Além disso, iremos oferecer materiais com questões acerca dos conteúdos trabalhados de forma que os usuários exerçam uma boa prática, todos esses pontos são separados em seções afim de tornar mais prática e rápida a navegação dos usuários pelo site.
